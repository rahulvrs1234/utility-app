Android Utility App

A simple Android application built using Jetpack Compose that provides daily utilities like weather updates and reminders.

Features

🌦 Live Weather Information

⏰ Reminder Notifications

🌙 Dark Mode Support

📱 Modern UI with Jetpack Compose

🔔 Alarm-based Reminder System

Technologies Used

Kotlin

Jetpack Compose

MVVM Architecture

Open-Meteo Weather API

Android AlarmManager

Notification Manager
